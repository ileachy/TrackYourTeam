## <img id="about" src="https://github.com/teamjuli0/readme-badges/blob/main/themes/clean-dark/menu-categories/about.png?raw=true" style="height: 40px"\

TrackYouTeam is a MySQL application designed to allow you to see and manage data about your team, be that in an office or outside setting!

## <img id="usage" src="https://github.com/teamjuli0/readme-badges/blob/main/themes/clean-dark/menu-categories/usage.png?raw=true" style="height: 40px">

To utilize this app simply clone the repo and do an "npm i" in the console to install the required packages, after that run the schema and seeds using the mysql termainal commands "source db/schema.sql" and "source db/seeds.sql".
Finally run node index and your program should start.
